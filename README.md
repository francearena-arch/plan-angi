# Angela – Nachhaltiger Lifestyle-Plan

Ein interaktiver, persönlicher Ernährungs- und Routinen-Plan.

## Öffnen

Einfach `index.html` im Browser öffnen – kein Server nötig.

Oder via GitHub Pages hosten:
1. Repository auf GitHub erstellen
2. Dateien hochladen
3. Settings → Pages → Branch: main → Save
4. Plan ist dann unter `https://[dein-username].github.io/[repo-name]` erreichbar

## Inhalt

- Ernährungsplan (Bürotag / Heimtag / Wochenende)
- Batch Cooking Anleitung
- Supplement-Übersicht
- Routinen & Wochenrhythmus
- Interaktiver Weekly Check-in

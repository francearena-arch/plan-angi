# Angela – Nachhaltiger Lifestyle-Plan (v2)

Ein interaktiver, persönlicher Ernährungs- und Routinen-Plan – angepasst an Angelas echten Alltag.

## Öffnen

Einfach `index.html` im Browser öffnen – kein Server nötig.

## GitHub Pages

1. Dateien ins Repository hochladen (index.html + README.md)
2. Settings → Pages → Branch: main → Save
3. Plan erreichbar unter `https://[username].github.io/[repo-name]`

## Was neu ist in v2

- Navigation mit 5 Tabs: Ernährung / Meal-Prep / Supplements / Routinen / Weekly Check-in
- Ernährungsplan angepasst an Angelas echten Wochenrhythmus (Bürotage Di–Do, H/O Mo, Sport-Zeiten)
- Yoga-Abende (Mo + Di 20:30, Mi 18:15) berücksichtigt → leichter Snack statt Abendessen
- Business Lunch als eigene Mahlzeit-Option
- Weekly Check-in mit Verlaufshistorie, Chart und Export/Import (JSON)
